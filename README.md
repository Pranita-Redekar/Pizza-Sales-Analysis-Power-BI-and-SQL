<p align="center">
An End-to-End Data Analytics Project using <b>SQL</b>, <b>Power BI</b>, <b>DAX</b>, and <b>Power Query</b>
</p>

<hr>

<h2>📖 Project Overview</h2>

<p>
The <b>Pizza Sales Analysis Dashboard</b> is an end-to-end Business Intelligence project that analyzes pizza sales data to uncover valuable business insights.
Using SQL for data analysis and Power BI for visualization, this project highlights sales performance, customer purchasing patterns, revenue trends, and product performance through an interactive dashboard.
</p>

<hr>

<h2>🎯 Project Objectives</h2>

<ul>
  <li>Analyze overall sales performance and business growth.</li>
  <li>Monitor key business KPIs.</li>
  <li>Identify best-selling and least-selling pizzas.</li>
  <li>Understand customer ordering behavior.</li>
  <li>Compare sales across pizza categories and sizes.</li>
  <li>Create an interactive dashboard for business decision-making.</li>
</ul>

<h2>⚙️ Project Workflow</h2>

<h3>📌 1. Data Preparation</h3>

<ul>
  <li>Imported and explored the pizza sales dataset.</li>
  <li>Validated data quality and column information.</li>
  <li>Prepared the dataset for analysis.</li>
</ul>

<h3>📌 2. SQL Business Analysis</h3>

<ul>
  <li>Calculated Total Revenue, Total Orders, Average Order Value, and Average Pizzas per Order.</li>
  <li>Analyzed daily and monthly sales trends.</li>
  <li>Compared sales by pizza category and size.</li>
  <li>Identified Top 5 and Bottom 5 selling pizzas.</li>
  <li>Generated business insights using SQL queries.</li>
</ul>

<h3>📌 3. Power BI Dashboard Development</h3>

<ul>
  <li>Created KPI cards using DAX measures.</li>
  <li>Designed interactive charts and visualizations.</li>
  <li>Added slicers for Month, Pizza Category, and Pizza Size.</li>
  <li>Built an easy-to-use dashboard for business reporting.</li>
</ul>

<hr>

<h2>📊 Dashboard Features</h2>

✅ KPI Cards (Revenue, Orders, Average Order Value, Average Pizzas per Order)

✅ Monthly Revenue Trend
